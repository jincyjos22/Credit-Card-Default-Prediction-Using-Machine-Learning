Machine Learning based Credit Card Default Prediction

📌 Project Overview

Credit card default is the important financial risk prediction problem to financial institutions given that it causes huge monetary losses. This project is about creating a machine learning model which can predict if a customer will default his/her credit-card payment next month.

The model assists in identifying at-risk customers to enable strategic decision-making by analyzing customer data like payment history, bill statements and demographic information.

🎯 Objective

Build and assess several machine learning models to predict the risk of a credit card issuer suffering from default at an accurate level, so to choose the best model.

📊 Dataset Information

Source: UCI Machine Learning Repository
Link:https://archive.ics.uci.edu/dataset/350/default+of+credit+card+

Dataset: Default of Credit Card Clients

🔹 Features:

LIMIT_BAL – Credit limit

SEX – Gender

EDUCATION – Education level

MARRIAGE – Marital status

AGE – Customer age

PAY_0 to PAY_6 — Repayment status (last 6 months)

BILL_AMT1:BILL_AMT6 — Amount of each bill

PAY_AMT1 – PAY_AMT6 Payment amounts

🎯 Target Variable:

default.payment.next.month

1 = Default

0 = No Default
